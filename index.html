Tentang Saya

Halo! Saat ini saya bertugas di Direktorat Jenderal Pajak (DJP), Indonesia, sambil melanjutkan pendidikan di Politeknik Keuangan Negara STAN (PKN STAN). Perjalanan profesional saya dimulai dengan empat tahun yang berharga mendapatkan beragam pengalaman dalam layanan pelanggan, dukungan TI, dan memberikan bantuan kesekretariatan untuk pejabat Eselon III di sektor publik.

Didorong oleh keingintahuan akan potensi teknologi untuk meningkatkan proses dan menghasilkan wawasan, saya aktif menyelami dunia pemrograman (HTML, CSS, JavaScript) dan meningkatkan keterampilan saya dalam analisis data. Saya mahir dengan aplikasi Microsoft Office, terutama Excel untuk manipulasi data, dan saya sedang belajar membuat visualisasi yang menarik dengan Power BI.

Saya bersemangat untuk menggabungkan latar belakang administrasi publik saya dengan keterampilan teknis yang sedang berkembang untuk menyumbangkan solusi inovatif dan mendorong efisiensi. Saya selalu bersemangat untuk belajar dan berkolaborasi.

Proyek & Perjalanan Belajar

Visualisasi Data: [Panola Board: Aplikasi Kepatuhan Berbasis Wilayah]

Memanfaatkan Power BI untuk menganalisis kepatuhan wajib pajak berdasarkan wilayah dan membuat dasbor interaktif yang memvisualisasikan tren dan pola utama terkait tingkat kepatuhan berdasarkan sektor dan wilayah. Berfokus pada penceritaan visual yang jelas.

Teknologi: Power BI

Keahlian Excel: [Aplikasi Automatisasi SPT 1770 EFORM]

Mengembangkan solusi MS Excel tingkat lanjut, memanfaatkan formula kompleks dan VBA untuk mengotomatiskan input dan pemrosesan EFORM SPT 1770, secara signifikan mengurangi upaya manual dan meningkatkan akurasi.

PDF Teknologi: MS Excel-VBA

Latihan Coding: Portofolio Pribadi

Membangun situs web portofolio pribadi ini dari awal sebagai latihan pembelajaran praktis dalam pengembangan front-end. Menerapkan konsep dasar HTML, CSS, dan JavaScript, termasuk desain responsif dan manipulasi DOM dasar.

Kode Teknologi: HTML, CSS, JS

Terus belajar dan mengeksplorasi cara menerapkan keterampilan teknis secara efektif.
